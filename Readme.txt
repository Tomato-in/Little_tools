学生端提交方法：
1. 所有作业应为word版本，文件名后缀为'.docx'；
2. 使用“薄荷图床”上传你的作业，获取链接；
3. 将你的相关信息和作业链接填写到课代表发到群里的表单中。

注：关于3中提到的表单，课代表去了解WPS表单功能。

课代表下载方法：
1. 整理表单，同模板保持格式一致；
2. 把脚本和表单放到一个文件夹中；
3. 获取共享文档的绝对位置，注意去掉双引号；
4. 点击collection脚本，输入第3步获取的位置，点击回车运行。

注意事项！！！
1. 薄荷图床无需注册，非会员链接有效期为3天，文件上限为5M；
2. 如果收集文件格式较多，可以压缩为'.zip'格式。
